Instructions

Simply click the "Source Code" link to download the portal.zip file available in this directory. Unzip it into a directory, and navigate to apps/myApps/visualizer and open "visualizer.xcodeproj" to open the project. Once it is open, click the play button to run the app. Alternatively, navigate to apps/myApps/visualizer/bin/ and choose the "visualizer" executable and run it.
